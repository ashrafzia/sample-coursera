# Contributing Guidelines

Thanks for your interest in contributing! This document explains how to fork/branch/commit and open pull requests, plus the style rules we follow.

## 1) How to contribute (fork → branch → PR)

1. **Fork** the repo to your GitHub account.
2. **Clone** your fork and set upstream:
   ```bash
   git clone https://github.com/<your-username>/<repo>.git
   cd <repo>
   git remote add upstream https://github.com/<origin-owner>/<repo>.git
